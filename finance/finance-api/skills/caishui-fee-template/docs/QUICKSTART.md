# 快速开始指南

## 5分钟上手

### 1. 安装依赖（1分钟）

```bash
pip install pandas requests websocket-client openpyxl
```

### 2. 启动浏览器（1分钟）

**macOS:**
```bash
/Applications/Microsoft\ Edge.app/Contents/MacOS/Microsoft\ Edge \
  --remote-debugging-port=9223
```

**Windows:**
```cmd
"C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --remote-debugging-port=9223
```

### 3. 登录系统（2分钟）

1. 在打开的 Edge 中访问：https://cst.uf-tree.com
2. 输入账号密码登录
3. 进入：单据设置 → 费用模板
4. **保持页面打开**

### 4. 准备Excel（1分钟）

复制 `examples/费用模板示例.xlsx`，修改内容：

| 一级科目 | 二级科目 |
|---------|---------|
| 差旅费 | 飞机票 |
| 差旅费 | 火车票 |
| 办公费 | 文具 |

### 5. 运行脚本

```bash
python3 scripts/add_fee_templates.py /path/to/你的Excel.xlsx
```

## 常见问题

### Q: 提示 "Connection refused"
**A:** Edge 浏览器未启动调试模式，请参考上方步骤2

### Q: 提示 "未找到财税通页面"
**A:** 请确保已登录并打开了费用模板页面

### Q: 提示 "未找到一级科目"
**A:** Excel中的一级科目名称必须与系统中的完全一致

### Q: 添加成功但字段没有继承
**A:** 这是正常的显示问题，实际字段已继承。用 `getFeeTemplateById` 查询能看到完整字段
